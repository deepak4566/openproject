Color Variables
===============

## Global Color Variables

```
@colors
- $global-font-color - -
- $header-bg-color - $header-logo-bg-color -
```

## Button Color Variables

```
@colors

- $button-font-color           - $button-font-hover-color           -
- $button-highlight-font-color - $button-highlight-font-hover-color -
```
